Student Management Portal (GUI)

A JavaFX desktop application with real-time cloud integration via Supabase.

Prerequisites
JDK: 17 or higher

Internet: Required for cloud database connectivity

JavaFX Portal Features:

Multi-View UI: Seamless navigation between Login, Dashboard, and Registration screens.

Live Cloud Sync: Real-time data storage and retrieval via Supabase (PostgreSQL).

Data Integrity: Dropdown menus for Program and Year Level to prevent entry errors.

Professional Branding: Custom PUP-themed interface (#800000 / #FFDF00).

Error Handling: Active UI feedback for duplicate Student IDs and connection issues.
